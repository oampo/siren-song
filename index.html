SIREN SONG

ABOUT SIREN SONG

Siren Song is an ∞ audio-visual game in which players guide Sirens through an endless algorithmically generated level. As the Sirens navigate the level they produce glitchy clouds of colour and hypnotic 8-bit drones.

Siren Song uses WebGLet and the HTML5 canvas to create the graphics, and Audiolet to generate and schedule the sounds.

TIPS
  • No visuals? You need a WebGL enabled browser. See http://get.webgl.org
  • No sound? Use a Dev build of Chrome with the Web Audio API enabled in about:flags.
  • Prefer Firefox? Versions >= 4 should give you working sound.
  • Hit F11. Siren Song looks better in fullscreen.
  • Guide the sirens to hear their song.
  • The controls? Left/Right or A/D.
  • Too easy? Keep it up! It gets harder.
  • Too hard? Let everything drift for a while.
  • Want a challenge? 100,000 points is tough.
  • Have fun!
